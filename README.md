## Sierpinski Triangle Emulator

A simple [Sierpi≈Ñski triangle](https://en.wikipedia.org/wiki/Sierpi%C5%84ski_triangle) Emulator, powerd by [Iced](https://github.com/iced-rs/iced).

Left-click add fixed point, right-click remove fixed point.
